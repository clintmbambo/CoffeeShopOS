# CoffeeShopOpsDemo

## Overview

CoffeeShopOpsDemo is a **demo web application for coffee shop operations**, focusing on **stock tracking** and **sales/profit visibility**. This system helps coffee shop owners quickly see low-stock alerts, monitor product profitability, and visualize overall profit in a simple, intuitive interface.

Built with **ASP.NET Core MVC**, **C#**, **Razor Pages**, **Tailwind CSS**, and hosted on **Azure**.

## Features

* **Stock / Inventory Tracking**

  * Add, edit, delete products
  * Track stock inflow and outflow
  * Low-stock alerts

* **Profit / Sales Visibility**

  * Record sales transactions
  * Calculate profit per product and total profit
  * Simple visual charts for quick insight

## Tech Stack

* ASP.NET Core MVC
* C#
* Razor Pages
* Tailwind CSS
* Azure Web App + Azure SQL (demo deployment)

## Demo Data

The application includes **dummy data** for milk, beans, syrups, and other common coffee shop products to illustrate functionality.

## Setup Instructions

1. Clone the repository

```bash
git clone https://github.com/<username>/CoffeeShopOpsDemo.git
```

2. Open in Visual Studio 2022
3. Restore NuGet packages
4. Configure your SQL Server / Azure SQL connection string in `appsettings.json`
5. Run the application
